# java-employee-payroll-system
A console-based Employee Payroll Management System developed in Java. This project demonstrates the use of interfaces, encapsulation, arrays of objects, salary calculations (HRA &amp; DA), and a menu-driven program structure using core OOP principles.

# Java Employee Payroll System

## Features
- Add Employee
- Calculate Salary (HRA 20%, DA 10%)
- Display Employee Details
- Menu-driven system
- Uses Interface for Payroll operations

## Concepts Used
- Interface
- Encapsulation
- Arrays of Objects
- Constructors
- Switch-case
- Looping (do-while)

## Technologies
- Java
- VS Code
